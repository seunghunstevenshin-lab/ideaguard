법적 활용 가이드

IdeaGuard가 제공하는 것과 그 한계, 그리고 법적 효력을 강화하는 방법을 솔직하게 안내합니다.

IdeaGuard가 제공하는 것

  • SHA-256 해시 + 서버 타임스탬프 — 수학적으로 조작 불가능한 착안일 기록
  • 원본 비공개(Zero-Knowledge) — 해시값만 저장, 원본 내용은 서버에 없음
  • NDA 자동 생성 + 수령자 서명 — 공동창업 전 신뢰 확보
  • 도덕적 억제력 — "기록이 있다"는 사실만으로도 분쟁 발생 억제
  • 증명 부담 전환 — 해시 기록이 있으면 상대방이 "기록이 조작됐다"는 것을 직접 증명해야 합니다. 내가 증명하는 것보다 훨씬 유리한 법적 위치입니다.

솔직한 한계

  • 한국 전자서명법상 공인 타임스탬프 기관(TSA) 인증서가 아닙니다
  • 법원 제출 시 단독으로는 법적 추정력이 발생하지 않습니다
  • 특허권·저작권·상표권 등록을 대체하지 않습니다

하지만 아래 방법으로 법적 효력을 크게 강화할 수 있습니다.

법적 효력 강화 3단계

  1. 현재

    IdeaGuard 해시 등록

    SHA-256 해시 + 서버 타임스탬프로 착안일 기록. 도덕적 억제력 확보.

    법적 강도 ●●○○○
  2. 추천

    특허청 영업비밀 원본증명 추가 등록 무료

    IdeaGuard에서 생성한 해시값을 복사하여 특허청 영업비밀 원본증명 서비스에 동일하게 등록합니다. 공인 정부 기관 기록 = 법적 추정력 발생.

    특허청 원본증명 바로가기 법적 강도 ●●●●○
  3. 추가

    NDA 체결

    공동창업자 · 투자자에게 아이디어를 공유할 때 IdeaGuard NDA를 체결합니다. 서명 기록 + IP 주소 + 서명 일시가 자동 저장됩니다.

    NDA 작성하기 법적 강도 ●●●●●
